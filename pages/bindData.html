Bind Data

Now, it's time to bind object to the svg element. Let's rock!

Bind Text

Add flexjs-bind-target attribute to the svg:text element which you want to bind. Set value as the attribute name which mapped to the object.

First of all, you should prepare a svg element as template. For example, follow <g>...</g> is the template svg element.

<svg version="1.1" xmlns="http://www.w3.org/2000/svg">
    <g id="tmpl">
        <text>Hello FlexJS</text>
    </g>
</svg>

Then use function flexjs to generate as template object. Follow tmpl is template object.

var svg = document.getElementById("tmpl");
var tmpl = flexjs(svg);

Use getInstance generate an svg copy.

var ele = tmpl.getInstance();

And append copy to your target svg container.

<svg id="cntr" version="1.1" xmlns="http://www.w3.org/2000/svg">
</svg>

var cntr = document.getElementById("cntr");
cntr.appendChild(ele);
