Auxiliary

FlexJS provide some auxiliary function to fast your svg opertaion.

.addClass(className)

Add an class in to element.

.removeClass(className)

Remove an class of the element.

.addTrans(key, value)

Add tansform attribute in to element.

ele.addTrans("translate", "5, 10");
// Same as:
ele.addTrans("translate", [5, 10]);

.removeTrans(key)

Remove tansform attribute of the element.

ele.removeTrans("translate");

.trans(args)

Get / Set tansform attributes in to element. args as empty will not set. Return transform as object.

.attr(key[, value])

Get / Set attribute in to element. args as empty will not set. Return attribute value.

ele.attr("fill", "#FFF");

.find(filter)

Find sub-elements of the element as Array.

ele.find("g > text");

.update()

Update element by bind data.

.object(args)

Get / Set bind object to the element. args as empty will not set. Return binded object.

.object(key[, value])

Get / Set bind object attribute to the element. args as empty will not set. Return binded object attribute.
